# Noble Dancing Game Companion App

A thematic website companion for the Noble Dancing Game, allowing players to select couples and scenarios, view rules, and manage gameplay elements.

## Features

- **Front Page**: Elegant landing with PLAY button.
- **Selection Page**: Choose your main couple and scenario with hover/click for details.
- **Gameplay Page**: View initiative, agent cards, actions, scenario info, trackers, and generate round events.
- **Rules Page**: Book-like layout with table of contents and sections.
- **Responsive Design**: Looks good on phone and computer.
- **Popups**: Click on names to see descriptions.

## Setup

Open `index.html` in a web browser to start.

## Dummy Data

- 4 Couples with stats and abilities.
- 3 Scenarios with objectives and rules.
- 6 Events, one replaced by scenario event.

## Customization

Replace placeholder images in `images/` folder.
